## How to use

1. Clone repo to ~/.emacs.d

```bash
git clone https://github.com/jessekelly881/.emacs.d ~/.emacs.d
```

2. Run emacs

```bash
emacs
```

**Warning** If you already have an emacs config be sure to back it up first.
